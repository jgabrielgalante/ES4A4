# Comente Sobre

## Desenvolvimento

```
git clone https://github.com/jgabrielgalante/ES4A4.git
```

### Front-end

- NPM 8.5
- NodeJS 16.15

```
cd client
npm i
npm run dev
```

### Back-end

Deployment: https://comente-sobre-api.herokuapp.com/

- Python 3.10
- Poetry 1.1

```
cd ../server
poetry install
poetry shell
python manage.py runserver
```
